# Pasapalabras Bíblico

Juego de Pasapalabras con temática bíblica, diseñado para jugar en grupos. Funciona en cualquier celular desde el navegador. Diseño responsive optimizado para pantallas pequeñas.

## Cómo jugar

1. Ingresá la cantidad de grupos (2 a 8) y sus nombres
2. Cada grupo tiene su propio rosco con preguntas diferentes
3. El grupo activo responde su letra actual:
   - **✓ Correcto** → suma un punto y sigue con la siguiente letra
   - **⇒ Paso** → marca la letra en naranja, pasa al siguiente grupo
   - **✗ Incorrecto** → marca la letra en rojo, pasa al siguiente grupo
4. Las letras en paso vuelven a aparecer cuando el grupo completa la vuelta
5. Gana el grupo con más respuestas correctas al finalizar todos los roscos

## Letras incluidas

22 letras: A B C D E F G H I J K L M N O P R S T U V Z

Se omiten las letras Q, W, X e Y. La letra K usa términos del griego bíblico:
- **K** → Kerygma, Koinonía, Kyrios
